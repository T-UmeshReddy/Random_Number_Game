# 🎯 Guess The Number Game

A simple and interactive Guess The Number web game where the player tries to guess a randomly generated number between 1 and 100 within limited chances.

This project is built using HTML, CSS, and JavaScript and provides instant feedback to the user after each guess.

## 📸 Project Overview

This game allows users to guess a number between 1 and 100. The system generates a random number, and the player has limited chances to guess it correctly.

Features of the interface:
- Input field to enter a number
- Guess button to submit the number
- Feedback messages for each guess
- Display of remaining chances

## 🛠 Technologies Used

- HTML5
- CSS3

## 🎮 How the Game Works

1. The system randomly generates a number between 1 and 100.
2. The user enters a number in the input field.
3. The game checks the guessed number and provides feedback:
   - Too Low
   - Too High
   - Correct Guess
4. The player has 5 chances to guess the correct number.
5. If the correct number is guessed, the player wins the game.
6. If all chances are used, the game ends.
